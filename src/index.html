Samples overview

Welcome to the samples overview for OpenLayers Base Packages. These packages are intended to be used with the Open Pioneer Trails framework. The source code for the samples can be found in the src/samples/ directory.

Samples

  • Showcase
    This application demonstrates the main components of the OpenLayers Base Packages.
  • Sample App
    This application combines multiple components of the OpenLayers Base Packages. It can be used as a starting point for new applications.
  • OGC API Features and OGC API Tiles
    Demonstrates the usage of an "OGC API Features" and "OGC API Tiles" layer in the map.

Test applications

These apps are primarily used for testing.

  • Basemap switcher
    Test app for the basemap switcher.
  • Result List
    Test app for the result list.
  • Printing API
    Test app for the printing api.
  • TOC + Health Check
    Demonstrates the TOC and health check to detect unavailable services.
  • Highlight and zoom
    Test app the map model's highlight and zoom API.
  • Menu fix
    Test app to verify that Chakra's Menu component works correctly inside our Shadow DOM setup.
