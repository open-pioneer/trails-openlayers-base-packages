Samples overview

Welcome to the samples overview for OpenLayers Base Packages. These packages are intended to be used with the Open Pioneer Trails framework. The source code for the samples can be found in the src/samples/ directory.

Samples

  • Default
    This app shows a sample setup of the main features.
  • Authentication
    Sample app for the authentication package.
  • Notifier
    Demonstrates how to use notifications.
  • OGC API - Features
    Demonstrates the usage of an "OGC API - Features" layer in the map.
